GROM创建联合索引

之前提到一个联合约束，那么根据需求再次做一个演示：

type Demo struct {
	ID           uint          `gorm:"primary_key"`
	CreatedAt    time.Time
	UpdatedAt    time.Time
	DeletedAt    *time.Time    `gorm:"index;unique_index:name_d"`
	Name     string            `gorm:"unique_index:name_d"`
	Status       int         
}

通过demo 迁移后，deleted_at 与 name 会形成一个联合约束。
